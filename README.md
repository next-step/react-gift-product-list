# react-gift-login

- 주문하기 페이지의 폼을 React Hook Form 로 리팩터링
  - (form 데이터를 useState로 다루는 경우는 없게 해요.)
- 받는 사람 기능 고도화
  - 참고 URL을 살펴보고 받는 사람 기능의 UI를 구현
  - 받는 사람을 최대 10명까지 등록 가능
  - 10명의 정보가 정확하게 입력되지 않으면 정보가 반영되지 않음
  - 받는 사람의 전화번호가 중복 안됨.
  - 최소 선택 수량은 1개 이상
  - 전화번호는 01012341234 형태만
- 주문하기 페이지
  - 단체 주문 리스트 표시
