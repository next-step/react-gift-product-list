## Step 2. 주문하기 페이지 구현
### 로그인 정보
- [x] 로그인 정보 저장할 수 있게 세션 스토리지에 저장
- [x] 로그인 한 유저 로그인 버튼 클릭 시 마이페이지(/my)로 연결
- [x] 로그인 하지 않은 사람이 마이페이지에 접속하면 로그인 페이지로 연결
- [x] 마이페이지에서 로그아웃 시 로그인(/login) 페이지로 연결

### 주문하기
- [x] 상품 아이템을 클릭 시 주문하기 페이지로 연결되도록 구현
- [x] 만약 로그인이 되어있지 않다면 로그인 페이지로 이동 -> 로그인 성공 시 주문하기 페이지로 돌아오기
- [x] 주문하기 버튼 클릭 시 유효성 검사 조건이 충족했는지 확인
