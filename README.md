## 3단계 과제 구현 내용

- 로그인 폼과 주문 폼을 react-hook-form 라이브러리를 이용해 리팩터링했습니다.
- 유효성 검사 로직에 zod 라이브러리를 적용시켜 가독성과 유지보수성을 높였습니다.
- 2단계 마지막 피드백 내용 적용 했습니다.
- 단체주문 로직을 구현했습니다. 모달을 통해 여러명의 Receiver를 설정할 수있게 했습니다.
- 모달 내부에서 새로운 useForm을 이용해 단체주문 로직을 구현했던것을 OrderPage.tsx에서 useForm을 확장시켜 사용하여 내부의 모든 컴포넌트에서 사용할 수 있도록 했습니다.
- 모달을 통해 받는사람 목록을 수정할 수 있게 했습니다.
