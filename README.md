## 3단계 과제 구현 내용

- 1단계 시에 과제 내용을 착각해 일부 구현했던 상품 테마 리스트 UI를 완성시켰습니다.
- 선물 테마 섹션에서 api요청시에 404에러 발생 시 홈화면으로 리디렉트 기능, 기본적인 에러 핸들링 구현했습니다.
- 선물 테마 섹션에서 /api/themes/:themeId/info, /api/themes/:themeId/products 를 둘다 요청하던 구조를, /api/themes/:themeId/products는 선물 테마 리스트 컴포넌트 안으로 분리했습니다.
- 선물 테마 리스트에 무한 스크롤 기능을 구현했습니다.
- 선물 테마 리스트의 상품 카드를 클릭하면 해당 상품의 주문 페이지로 이동하는 로직을 추가했습니다.
- 선물 테마 리스트에서의 api요청에 대한 기본적인 에러핸들링 로직 구현했습니다.
