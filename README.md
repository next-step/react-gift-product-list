# react-gift-order

## 1단계 과제 진행 요구사항

- [x] ID
  - [x] ID는 반드시 입력되어야 함
  - [x] ID는 이메일 형식이어야 함
  - [x] 조건이 충족되지 않으면 에러메시지 띄우기
  - [x] 조건 충족 시 에러메시지 사라짐
- [x] PW
  - [x] 최소 8글자 이상
  - [x] 조건이 충족되지 않으면 에러메시지
- [x] ID와 PW 조건이 모두 충족되면 버튼이 활성화

## 2단계 과제 진행 요구사항

- [x] 로그인 정보
  - [x] 로그인 정보를 어디서든 사용할 수 있게하기
  - [x] 새로고침을 하더라도 로그인 정보 유지
  - [x] 로그인 한 유저는 로그인 버튼 클릭 시 마이페이지로 연결
  - [x] 로그인하지 않은 사람이 마이페이지 접속 시 로그인 페이지로 연결
  - [x] 마이페이지에 로그아웃 가능. 로그아웃 시 로그인 페이지로 연결
- [ ] 주문하기
  - [x] 상품 아이템을 클릭하면 주문하기 페이지로 연결
  - [x] 로그인이 되어있지 않다면 로그인 유도(로그인 페이지로 연결), 로그인 성공 시 주문하기 페이지로 접근 가능하게 함
  - [x] 버튼 클릭 시 아래 유효성 검사 조건이 충족되지 않으면 각 폼 필드에 안내 메시지를 보여줌
    - 메시지는 반드시 입력되어야 함
    - 보내는 사람 일므이 반드시 입력되어야 함
    - 받는 사람 이름이 반드시 입력되어야 함
    - 받는 사람 전화번호가 반드시 입력되고 전화번호 규칙에 맞아야 함(01012341234)
    - 수량은 1개 이상
