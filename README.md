## 1단계 과제 구현 내용

- API 환경 변수를 설정해 유지보수성과 보안수준을 높였습니다.
- useApiRequest.ts 커스텀훅으로 API 요청 로직 구현했습니다. (객체로 받을 수 있게 해서 급상승 선물 랭킹같은 여러개의 요청 설정에 대응할 수 있게 했습니다.)
- 선물 테마의 리스트를 API요청으로 받아온 목록으로 구성합니다.
- 원래 자체 컴포넌트 상태로만 유지했던 급상승 선물 랭킹 버튼들을 쿼리 스트링 기반 상태 관리 방식으로 리팩터링했습니다.
- 버튼을 통한 쿼리 스트링을 통해 API 요청을 해 급상승 선물 랭킹 리스트를 구현했습니다.
- 급상승 선물 랭킹 리스트의 상품 카드 왼쪽 상단의 랭킹표시를 구현했습니다.

이후 단계 과제인 선물 테마 섹션 이미지 클릭 후 나오는 목록을 과제를 착각해 조금 구현했습니다.

## 1단계 피드백 적용 내용

- API url 엔드포인트를 API_ENDPOINTS.ts 파일로 추출하여 상수로 관리해 유지보수성을 높였습니다.
- api_types.ts 파일을 통해 api요청 관련 타입들을 모아서 필요한 컴포넌트에서 import해서 사용할 수 있게 했습니다.
- useApiRequest 커스텀 훅을 GET 요청만 가능한 상태에서 POST/PUT 등 다른 HTTP메서드도 사용할 수 있도록 확장했습니다. (기본값은 GET요청)
